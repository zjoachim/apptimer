# 使用指南

## 安装与启动

AppTimer 是一个**便携式应用**，无需安装：

1. 下载 `AppTimer.exe`
2. 放到你喜欢的文件夹（建议 `D:\Tools\AppTimer\`）
3. 双击运行即可

首次启动后，程序会自动最小化到系统托盘（右下角图标），在后台静默记录你的程序使用情况。双击托盘图标或右键选择「显示窗口」即可打开主界面。

## 界面概览

主界面分为以下几个区域：

| 区域 | 说明 |
|------|------|
| 顶部标题栏 | 状态指示灯 + 今日追踪时长 + 深色模式开关 |
| 今日使用 | 当前日期各程序的使用时长列表，含分类标签和占比 |
| 饼图 + 柱状图 | 今日各程序占比饼图 + 近 7 天使用趋势柱状图 |
| 分类&目标 | 设置程序分类标签 + 每日使用目标上限 |

## 追踪面板

**今日使用**列表实时显示：

- **程序**：自动识别的窗口进程名（优先显示软件官方名称）
- **分类**：你为该程序设置的标签
- **今日**：今天累计使用了多长时间
- **累计**：所有历史数据中的总计使用时长
- **占比**：该程序占今日总使用时长的百分比

::: tip 闲置检测
当你离开电脑超过设定的时间阈值（默认 5 分钟），追踪器会自动暂停，不计入使用时长，确保数据准确。
:::

## 分类标签

在「分类&目标」标签页中：

1. 左侧列表选择要分类的程序
2. 右侧下拉框选择分类标签
3. 可选标签：**工作、学习、娱乐、社交、开发、设计、其他、未分类**

分类后，主面板和报表中都会按分类统计，图表也会按分类着色。

## 数据与报表

### 数据存储

所有数据保存在本地，路径为：

```
%USERPROFILE%\AppTimer\
```

按日期子文件夹组织，每天一个 JSON 文件。

### 查看历史

主界面下方的历史记录窗格可查看过往数据。

### 导出 CSV

点击「导出 CSV」按钮，可选择导出**今日数据**或**累计数据**，生成的文件可在 Excel 中打开分析。

### 自动清理

可在设置中配置数据保留天数（默认 90 天），过期数据自动删除，不必担心占用空间。

## 设置与自启动

### 深色模式

窗口右上角的开关可手动切换深色/浅色主题。程序启动时会自动跟随 Windows 系统主题设置。

### 开机自启动

勾选「开机自启动」后，AppTimer 会随 Windows 启动自动运行，无需手动打开。

### 最小化到托盘

关闭窗口时，程序不会退出，而是最小化到系统托盘继续后台追踪。右键托盘图标可选择「退出」彻底关闭。
