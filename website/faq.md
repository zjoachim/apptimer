# 常见问题

## 数据保存在哪里？

数据存储在 `%USERPROFILE%\AppTimer\` 目录下（通常是 `C:\Users\你的用户名\AppTimer\`）。

按日期子文件夹组织，每天一个 JSON 文件。分类标签和目标设置也保存在该目录。

## 如何迁移数据到新电脑？

直接复制 `%USERPROFILE%\AppTimer\` 整个文件夹到新电脑的同一位置即可，所有历史和设置都会保留。

## 深色模式怎么切换？

两种方式：

1. **手动切换**：主窗口右上角的深色/浅色开关
2. **跟随系统**：AppTimer 启动时自动检测 Windows 主题设置（设置 → 个性化 → 颜色）

## 为什么有些程序没有被记录？

AppTimer 只追踪有窗口标题的程序。后台服务、命令行窗口（无标题）、Metro/UWP 应用可能无法正确识别。

## 如何设置每日使用目标？

1. 切换到「分类&目标」标签页
2. 在右侧「每日目标」区域选择程序
3. 设定每日使用时长上限（小时/分钟）
4. 点击保存

当该程序当日使用时长超过目标时，会弹出提醒。

## 休息提醒是什么？

默认连续使用电脑超过 **45 分钟**，AppTimer 会弹出休息提醒。该阈值可在设置中调整。

## 需要联网吗？

不需要。AppTimer 完全离线运行，所有数据只在本地存储，不会上传任何信息。

## 杀毒软件报毒怎么办？

AppTimer 使用 PyInstaller 打包，部分杀毒软件可能对未签名的 exe 产生误报。可将程序所在文件夹加入杀毒软件白名单。如果仍有顾虑，可以直接用 Python 运行源码 `app_timer.py`。

## 支持哪些 Windows 版本？

Windows 10 及以上版本均可使用。
