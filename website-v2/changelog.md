# 更新日志

## v2.0

**发布日期**: 2026-05

### 架构变更

- 前后端分离：Python HTTP 服务 + Vue 3 前端
- Edge 无边框窗口：`--app` 模式启动，界面干净无地址栏
- 本地 HTTP 服务：`127.0.0.1:17777`

### 新增功能

- 7 个标签页：概览、累计、图表、趋势、分类、历史、设置
- ECharts 可视化：饼图 + 柱状图
- 趋势分析：近 7 天使用对比
- PDF 导出：带图表的报告
- 三级降级进程检测：提高窗口识别准确率
- 单实例互斥：防止重复启动

### 技术栈

- 前端：Vue 3 + ECharts + Tailwind CSS
- 后端：Python HTTP Server + Windows API
- 打包：PyInstaller one-file 模式

## v1.0

**发布日期**: 2026-05

### 功能

- 自动追踪 Windows 程序使用时长
- 程序分类标签（工作/学习/娱乐/社交/开发/设计/其他）
- 日/周/月报表自动生成
- 饼图 + 柱状图可视化（tkinter Canvas）
- 每日使用目标设定 + 超时提醒
- 休息提醒（连续使用 45 分钟）
- CSV 导出
- 深色模式（自动跟随 Windows 主题）
- 系统托盘 + 后台运行
- 开机自启动
