# 使用指南

## 安装与启动

AppTimer 是**便携式应用**，无需安装：

1. 下载 `AppTimer-v2.exe`（Win11）或 `AppTimer-v1.exe`（Win10）
2. 放到你喜欢的文件夹
3. 双击运行

启动后自动打开 Edge 无边框窗口，界面干净无地址栏。关闭窗口不会退出，程序继续在后台追踪。

::: tip Win10 用户
v2 使用 Edge 无边框窗口，Win10 上可能存在兼容问题。建议 Win10 用户下载 v1 版（tkinter 桌面窗口）。
:::

## 界面概览（v2）

v2 采用 Vue 3 + ECharts 架构，7 个标签页：

| 标签页 | 说明 |
|--------|------|
| 概览 | 今日使用时长、当前程序、实时状态 |
| 累计 | 所有历史程序的总使用时长 |
| 图表 | ECharts 饼图 + 柱状图可视化 |
| 趋势 | 近 7 天使用趋势对比 |
| 分类 | 程序分类标签管理（工作/学习/娱乐等） |
| 历史 | 按日期查看历史记录 |
| 设置 | 保留天数、自启动、深色模式等 |

## 追踪面板

**概览**标签页实时显示：

- **当前程序**：正在使用的窗口名称
- **今日时长**：今天累计使用总时长
- **程序列表**：各程序使用时长、分类标签、占比

## 分类标签

在「分类」标签页中：

1. 选择要分类的程序
2. 选择标签：**工作、学习、娱乐、社交、开发、设计、其他**
3. 分类后图表和报表中会按分类着色统计

## 数据与报表

### 数据存储

所有数据保存在本地桌面：

```
桌面\程序使用记录\
```

按日期子文件夹组织，每天一个 JSON 文件。

### 导出

- **CSV 导出**：导出今日或累计数据，Excel 直接打开
- **PDF 导出**：生成带图表的 PDF 报告

### 自动清理

可在设置中配置保留天数（默认 90 天），过期数据自动删除。

## 设置

### 开机自启动

在「设置」标签页开启，程序随 Windows 启动自动运行。

### 深色模式

设置中切换，自动跟随 Windows 系统主题。
