# 常见问题

## v1 和 v2 有什么区别？

| | v1 | v2 |
|---|---|---|
| 界面 | tkinter 桌面窗口 | Vue 3 + Edge 无边框窗口 |
| 图表 | tkinter Canvas | ECharts |
| 标签页 | 4 个 | 7 个 |
| 导出 | CSV | CSV + PDF |
| 系统 | Win10/11 | Win11 推荐 |
| 大小 | ~27MB | ~43MB |

Win10 用户建议用 v1，Win11 用 v2。

## 数据保存在哪里？

数据在桌面的 `程序使用记录` 文件夹中，按日期子文件夹组织。v1 和 v2 数据格式兼容，可以共用。

## 如何迁移数据到新电脑？

复制桌面的 `程序使用记录` 文件夹到新电脑桌面即可。

## 支持导出什么格式？

- **CSV**：纯数据表格，Excel 直接打开
- **PDF**：带图表的报告（仅 v2）

## 为什么有些程序没有被记录？

只追踪有窗口标题的程序。后台服务、无标题命令行窗口可能无法识别。

## 需要联网吗？

不需要。完全离线运行，数据只在本地存储。

## 杀毒软件报毒？

PyInstaller 打包的 exe 可能被误报。可加入白名单，或直接用 Python 运行源码。

## Edge 没装能用吗？

v2 优先使用 Edge，也支持 Chrome。如果都没有，自动用默认浏览器打开。

## Edge 下载时提示"通常不会下载"怎么办？

这是 Edge 对未签名 exe 的正常安全提示，不是病毒。点击底部弹窗的「**保留**」→「**仍然保留**」即可。如果担心安全性，可以在 [GitHub 仓库](https://github.com/zjoachim/apptimer) 直接下载源码用 Python 运行。
