# AppTimer

Windows 程序使用时间追踪器。自动记录每个程序的使用时长，帮你了解时间都花在了哪里。

## 下载

- [官网下载](https://zjoachim.github.io/apptimer/)
- [备用地址](https://zjoachim.github.io/apptimer/)

支持 Windows 10 / 11，无需安装，双击即用。

## 功能

**时间追踪**
- 自动记录前台程序使用时间，2 秒检测一次
- 空闲检测：离开电脑自动暂停计时
- 支持浏览器网站识别（B 站、YouTube、GitHub 等自动分类）

**数据管理**
- 分类标签：按程序归类（工作/学习/娱乐/其他）
- 每日目标：设定使用计划，超时提醒
- 累计统计：跨天自动汇总，数据防丢失
- 自动清理：过期数据按设定天数自动删除

**可视化**
- 今日概览：实时查看当前程序和使用时长
- 累计图表：柱状图 / 饼图展示使用分布
- 趋势分析：单程序多日使用趋势线
- 历史报告：按日期回溯每日使用记录

**导出与集成**
- PDF 报告导出
- CSV 数据导出（Excel 兼容）
- 开机自启动
- 系统托盘常驻

## 界面

深色主题，7 个功能页面：

| 页面 | 说明 |
|------|------|
| 概览 | 今日使用总览，当前程序，实时进度 |
| 累计 | 跨天累计统计，图表可视化 |
| 图表 | 使用分布柱状图 / 饼图 |
| 趋势 | 单程序多日趋势对比 |
| 分类 | 程序分类管理，每日目标设定 |
| 历史 | 按日期查看历史报告 |
| 设置 | 空闲阈值、保存间隔、自启动、卸载 |

## 数据存储

所有数据保存在桌面的 `程序使用记录` 文件夹，JSON 格式，可手动备份。
