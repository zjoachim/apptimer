数据功能 ​

  • 每日文件按日期子文件夹组织
  • 自动清理过期数据（可配置保留天数，默认 90 天）
  • 周报 / 月报自动生成
  • 使用趋势（跨天对比）
  • 导出 CSV（今日 / 累计）
  • 程序分类标签（工作 / 学习 / 娱乐 / 社交 / 开发 / 设计 / 其他）

  • 深色模式 / 浅色模式（自动跟随 Windows 系统主题）
  • 饼图可视化（今日各程序占比）

  • 每日使用目标设定 + 超时提醒
  • 程序连续使用提醒（默认 45 分钟休息提示）
  • 系统托盘图标 + 后台运行
  • 开机自启动选项
  • 实时状态指示灯
